Stage Objectif 3D - Réseau Social

Réalisation d'un réseau social interne

Durant mon stage chez Objectif 3D, j'ai développé un réseau social sur mesure destiné aux étudiants et aux enseignant. Ce projet a été conçu avec React JS pour le frontend et Node.js avec Express pour le backend.

Le système intègre une base de données MySQL pour la gestion des utilisateurs, publications, commentaires, et notifications. J'ai également utilisé Socket.IO pour la communication en temps réel (chat, notifications) ainsi que Fuse.js pour permettre une recherche intelligente des contenus comme les utilisateurs ou les publications.

Objectifs du stage

Concevoir une plateforme sociale privée pour faciliter les échanges entre les membres de l'école, permettre le partage de projets, de fichiers et la communication directe entre utilisateurs.

Technologies utilisées

  • React.js (frontend)
  • Node.js + Express (backend)
  • MySQL (base de données)
  • Socket.IO (temps réel)
  • Fuse.js (recherche intelligente)
Auncune image n'a été utilisée dans ce projet, car il s'agit d'un projet de stage interne à l'école Objectif 3D.
